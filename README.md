# Form Validation using HTML, CSS, and JavaScript

A simple and responsive **Form Validation project** built using **HTML, CSS, and JavaScript**.  
This project demonstrates how to validate user inputs on the client side to ensure correct data entry before submitting a form.

It checks inputs such as **name, email, password, and required fields**, displaying helpful error messages when the input is invalid.

---

## 🚀 Features

- Real-time form validation
- Error messages for invalid inputs
- Email format validation
- Password length validation
- Required field checking
- Clean and responsive UI
- Beginner-friendly JavaScript logic

---

## 🛠️ Built With

- **HTML5** – Form structure  
- **CSS3** – Styling and layout  
- **JavaScript (Vanilla JS)** – Validation logic

## ⚙️ How It Works

1. The user fills out the form fields.
2. JavaScript checks the inputs when the form is submitted.
3. If the input is invalid:
   - An error message appears.
   - The field is highlighted.
4. If all inputs are valid:
   - The form submission proceeds successfully.
